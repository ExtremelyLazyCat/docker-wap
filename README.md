Dockerized WAP/MMS Gateway for the Osmocom stack

Uses Mbuni + Kannel and a modified osmo-ggsn to create an MMSC + WAP Gateway usable by
clients on an Osmocom network.

Setup instructions are contained in SETUP.md, and important information on using MMS is
contained in the INFO.md file.

To clone this repository, run:
```
git clone https://github.com/ExtremelyLazyCat/docker-wap.git
```
